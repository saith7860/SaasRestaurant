README for SaaS Restaurant Project

# 🍔 SaaS Restaurant

A modern full-stack restaurant ordering web application built using React, Vite, Tailwind CSS, and backend APIs.

This project allows users to browse food categories, view menu items, add products to cart, and place food orders through a responsive mobile-friendly interface.

---

# 🚀 Features

## ✅ Authentication
- User Signup
- User Login
- Session-based authentication

## ✅ Restaurant Menu
- Categories display
- Dynamic food items
- Product variants
- Responsive UI

## ✅ Cart System
- Add to cart
- Quantity management
- Variant selection
- Cart badge counter

## ✅ Checkout System
- Order summary
- Delivery details form
- Place order functionality

## ✅ Responsive Design
- Mobile-first UI
- Tailwind CSS styling
- Optimized for all screen sizes
